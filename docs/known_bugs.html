Known Bugs

  • The server possibly miscounts number of dropped pieces / cleared lines.
  • Game-end announce (AnnounceScore) may lie about winlist rank (If A who has rank 3 with 10 points and B who has rank 4 with 9 points are on same team, they win a game and get 2 points, the following contradiction possible occurs: First, B has now 11 points, then the server announces B is rank 3. Second, A has now 12 points, then the server announces A is rank 3).
